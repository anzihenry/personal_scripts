# 个人脚本仓库

这个仓库包含了一些个人编写的脚本，用于各种自动化任务和工具。

## 目录结构

- `README.md` - 说明文档

## 使用方法

1. 克隆仓库到本地：
    ```bash
    git clone https://github.com/anzihenry/personal_scripts.git
    ```
2. 进入脚本目录：
    ```bash
    cd personal_scripts/scripts
    ```
3. 运行你需要的脚本，例如：
    ```bash
    python example_script.py
    ```
